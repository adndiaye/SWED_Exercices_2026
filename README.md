# SWED_Excercice_1

This repository contains the solutions and documentation for **Exercise 1** of the Software Engineering & Design (SWED) course.

## Contents

- **Task 1** – IDE installation documentation and 10 advantages of IDEs over plain text editors
- **Task 2** – Computational proof that the Ulam (Collatz) function terminates at 1 for all positive integers n < 1,000,000
- **Task 3** – This repository (GitHub account creation and first commit)
- **Task 4** – UML/CASE tool installation documentation
- **Task 5** – UML class diagram and object diagram for a `Developer` class, with Java implementation including setter integrity checks
- **Task 6** – Definition of software design and its distinction from software analysis
- **Task 7** – Justification of why software design is necessary in software projects
- **Task 8** – Analysis of the interdependence of architectural, database, UI, and component design activities
